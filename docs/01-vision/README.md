# Vision Documents

This directory contains the strategic vision and guiding principles for the Amplifier Usage Insights project.

## Contents

This directory should contain:

### VISION.md (Required)
The core vision document that defines:
- Project purpose and goals
- Target users and use cases
- Success criteria
- Strategic direction

Use the template in `docs/templates/VISION_TEMPLATE.md` to create this document.

### PRINCIPLES.md (Standard/Full tiers)
Core principles that guide decision-making throughout the project:
- Design principles
- Technical principles
- Values and constraints

Use the template in `docs/templates/PRINCIPLES_TEMPLATE.md` to create this document.

### SUCCESS-METRICS.md (Standard/Full tiers)
Quantifiable metrics to measure project success:
- Key performance indicators
- User satisfaction metrics
- Technical performance goals

Use the template in `docs/templates/SUCCESS_METRICS_TEMPLATE.md` to create this document.

Vision documents not yet created. Start with VISION.md as the foundation.
